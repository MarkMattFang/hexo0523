hexo使用問題

初次安裝Hexo建置部落格的時候，遇到不少奇怪的問題，多虧六角學院同學的幫忙下，才讓我能順利建置，在此分享我曾遇到的痛點。
特別推薦以下網址
CC的技術隨筆
常見問題

  1. PowerShell ISE初始設定都是Restricted，一定要變更才能後續動作
    CC的技術隨筆
  2. 建好檔案後一定要到指定位址，不然會失敗
    解決方案: cd ‘目的地’
  3. hexo clear && hexo deploy 部署問題

    解決方案: 不要使用”&&”，先跑hexo clear，再跑hexo deploy
    或&& 改成 ; (分號)也可以
  4. Deploy到GitHub沒反應或失敗
    解決方案: 在終端機的地方打以下兩個指令
    git config –global user.name
    git config –global user.email

2024 【六角學院】軟體工程師體驗營 心得

前言

早在元宇宙的出現，前公司安排講師介紹虛擬貨幣的時候，才發現這個不曾碰過的語言程式是可以成為我未來職業的可能，我也確實對虛擬的興趣更勝實體，又想起大學時想做遊戲的目標，加上這幾年來公司加薪的頻率少的可憐，工作量卻逐年爆發成長，身心倦怠之下，毅然裸辭準備專心學習!

心得

今年3月中跟前公司確定離職日後，我就開始搜尋能夠上課的課程，本意希望能無縫接軌直接系統性的學習，但看了下資策會、Tibame、政府就業通等的相關課程，時間銜接上都要超過1個多月，剩至要排到半年後，還需要全職當學生空窗這麼久，雖然知道是實體課程，需要按照日程表安排，但當前這些都不是我最佳的選擇，就在這個時候，我發現了六角學院的軟體工程師體驗營之後，就決定報名，等到4月11日可以報名的當下第一時間報名參加!!除了價格超佛之外，內容更是多到超乎預期，60小時的課程各章節有系統的規劃，最重要的是可以不受限時間、地點的學習，如果有問題，可以直接在Discord上發問，講師和同學都非常熱心的幫忙我解決這些問題，每個人都是從新手慢慢走來，學習的路上有六角學院的老師跟同學的幫助，我轉職的信心也大大增加，我非資工或資管的本科系，很多知識點真的無法一朝一夕的建立，學習確實需要時間&精力投入，六角學院的課就算是在職也能夠學習，特別推薦像我這樣想轉職各位，未來六角學院開的課我也是興趣滿滿想投入!!
